Data-Driven Workflow

  • Enemy HP, damage curves, wave timings, and reward pacing were tuned via spreadsheet passes.
  • Upgrade economy values were compared across multiple run lengths to avoid dead build paths.
  • Balancing notes tracked version-to-version changes so tuning decisions stayed intentional.
  • Team iteration stayed fast by decoupling content values from implementation details.
